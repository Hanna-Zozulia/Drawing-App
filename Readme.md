# Drawing App

---

## Описание

Приложение для рисования на холсте с возможностью сохранять, редактировать и удалять изображения. Реализовано на **TypeScript**, с использованием **Node.js** и **Express**.
Поддерживает:

- Рисование с выбором цвета и размера кисти
- Сохранение рисунков с указанием названия и цены
- Редактирование существующих рисунков
- Удаление отдельных рисунков или всей галереи
- Автоматическую загрузку галереи изображений с сервера

---

## Технологии

- Node.js
- Express  
- TypeScript

---

## Установка и запуск

1. Клонируйте репозиторий:

```bash
git clone 
cd chat
```

2. Установите зависимости:

```bash
npm install
```

3. Собираем проект:

```bash
npm run build
```

4. Запуск проекта:

```bash
npm start
```
